## Game with networking


### Installment requirements:

* git, 
* Cmake 3.16,
* yarn,
* node > 14
* python 3.8, pip3


### Install
* Run prepare.sh


### Note:

* clone read env in C
* yarn install
* install python env, pygame
* New implementation run only for all local or all non-local compared to node
* remember start node flag (-i)
